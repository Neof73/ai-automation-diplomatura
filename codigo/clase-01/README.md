# Clase 01 — ¿Qué automatizar? Mapeo de procesos y cálculo de ROI

## Archivos de esta clase

| Archivo | Descripción |
|---|---|
| `lumina-procesos-ejemplo.csv` | Mapa de los 5 procesos de Lumina con datos de tiempo y frecuencia |
| `mapa-procesos-template.csv` | Template vacío para completar con el negocio propio |
| `calculadora-roi.html` | Calculadora interactiva de ROI — abrir en cualquier navegador |
| `criterios-automatizacion.md` | El framework de 4 criterios para evaluar si un proceso se automatiza |

## Qué hacer en esta clase

1. Abrir `lumina-procesos-ejemplo.csv` en Google Sheets y analizar los datos de Lumina
2. Usar `calculadora-roi.html` para calcular el ROI del proceso más costoso de Lumina
3. Completar `mapa-procesos-template.csv` con los 3 procesos principales del negocio propio
4. Aplicar los criterios de `criterios-automatizacion.md` para priorizar

## Tarea para la semana

Completar el mapa de procesos con al menos 5 procesos del negocio propio, incluyendo:
- Tiempo real cronometrado (no estimado)
- Frecuencia real de la última semana
- Quién lo hace actualmente

El mapa completo se presenta en la clase 5 (Entregable 1).
