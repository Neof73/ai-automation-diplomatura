# Clase 02 — Hablarle a la IA: prompts que funcionan

## Archivos de esta clase

| Archivo | Descripción |
|---|---|
| `plantilla-riel.md` | El framework RIEL para construir prompts estructurados |
| `prompts-lumina-clase.txt` | Los 3 prompts de Lumina construidos durante la clase |

## Qué hacer en esta clase

1. Leer el framework RIEL en `plantilla-riel.md`
2. Analizar los prompts de Lumina en `prompts-lumina-clase.txt`
3. Resolver al menos 2 ejercicios de `ejercicios-prompts.md` durante la clase
4. Probar los prompts en ChatGPT (chat.openai.com) o Claude (claude.ai)

## Tarea para la semana

Construir 3 prompts para el negocio propio usando el framework RIEL:
- Uno para clasificar el tipo de mensaje recibido
- Uno para redactar una respuesta estándar a la consulta más frecuente
- Uno para resumir información de varios ítems en un reporte

Guardar los 3 prompts en un archivo de texto y traerlos a la clase 3.
