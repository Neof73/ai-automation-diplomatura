# Clase 04 — Tu primer Zap: automatización sin código con Zapier

## Archivos de esta clase

| Archivo | Descripción |
|---|---|
| `zap-lumina-descripcion.md` | Descripción paso a paso del Zap de Lumina para replicar |
| `webhook-payload-ejemplo.json` | Payload de ejemplo para probar el Zap con webhook |
| `email-confirmacion-template.html` | Template HTML del email de confirmación automática |
| `troubleshooting-zapier.md` | Soluciones a los problemas más comunes en Zapier |

## Qué hacer en esta clase

1. Crear cuenta en Zapier (zapier.com) si no la tenés
2. Seguir los pasos de `zap-lumina-descripcion.md` para construir el Zap de Lumina
3. Probar el Zap enviando un email de prueba al Gmail de Lumina
4. Verificar que la fila apareció en Google Sheets

## Prerrequisitos

- Cuenta de Gmail activa (puede ser la personal para la prueba)
- Planilla de Google Sheets lista con la hoja "Consultas" (clase 03)
- Cuenta de Zapier (plan gratuito alcanza para esta clase)

## Importante sobre el plan gratuito de Zapier

El plan gratuito incluye:
- 5 Zaps activos
- 100 tareas por mes
- Sin lógica condicional (eso lo vemos con n8n desde clase 06)

Para esta clase, el plan gratuito es suficiente.
