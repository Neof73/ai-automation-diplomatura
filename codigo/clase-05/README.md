## Qué presentar hoy

**Formato:** presentación oral de 5 minutos + Q&A de 2 minutos

**Lo que el grupo tiene que poder ver:**
1. El mapa de procesos del negocio (en pantalla o impreso)
2. Los 3 procesos priorizados para automatizar, con ROI calculado
3. El primer proceso seleccionado para el pipeline del Bloque 2

**No es necesario:**  tener el Zap funcionando perfectamente.
Es necesario: tener el análisis y la priorización claros.
