## El sistema completo del Bloque 3

Al terminar el Bloque 3, el sistema de Lumina tiene estos 5 componentes:

```
1. CAPTURA    → Gmail Trigger captura todos los emails automáticamente
2. CLASIFICA  → OpenAI clasifica el tipo de mensaje con comprensión del lenguaje
3. RESPONDE   → OpenAI redacta respuestas personalizadas con datos reales del pedido
4. DETECTA    → OpenAI analiza sentimiento y urgencia para priorizar reclamos
5. REPORTA    → Schedule Trigger genera el resumen ejecutivo todos los lunes
```
