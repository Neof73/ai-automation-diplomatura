# Clase 18 — ¿Cuánto vale lo que construiste? Medir el ROI real

## Archivos de esta clase

| Archivo | Descripción |
|---|---|
| `codigo/calcular-roi.js` | Script del nodo Code para calcular el ROI en tiempo real |
| `formulas-roi-sheets.md` | Fórmulas de Google Sheets para el dashboard de ROI |
| `supuestos-roi-template.md` | Template para documentar los supuestos del cálculo |

## Cómo usar estos archivos

1. Abrir `supuestos-roi-template.md` en VS Code y completar los supuestos propios
2. Implementar las fórmulas de `formulas-roi-sheets.md` en la hoja "Dashboard"
3. Agregar el nodo Code con `calcular-roi.js` al final del flujo del reporte
4. El reporte semanal ahora incluirá automáticamente el ROI calculado

## Los tres números que hay que citar de memoria para el Demo Day

1. **Mensajes procesados esta semana** (del dashboard)
2. **Tasa de resolución automática** (% sin intervención humana)
3. **ROI anual** (valor generado / costo del sistema)
