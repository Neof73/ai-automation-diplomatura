# Guion de presentación — Demo Day (10 minutos)

Leer este documento en VS Code durante los ensayos. En el Demo Day, tenerlo
abierto como referencia pero NO leer de él — hablar naturalmente.

---

## Minuto 0:00 — 1:00: EL ANTES

*"Antes de mostrar el sistema, cuento qué problema resuelve."*

**Completar:**
- El proceso que automaticé: [nombre del proceso]
- Antes, para hacer esto había que: [descripción de 2-3 pasos manuales]
- Tardaba: [X minutos/horas] y pasaba [X veces] por semana
- El principal problema era: [el costo, el tiempo, los errores, la inconsistencia]

---

## Minuto 1:00 — 2:00: EL SISTEMA EN UNA ORACIÓN

*"El sistema que construí hace esto:"*

**Completar (una oración, sin vocabulario técnico):**
"Cuando [evento que dispara el flujo], el sistema [qué hace automáticamente] y [resultado para el negocio]."

Ejemplo: "Cuando llega un email de cliente, el sistema lo clasifica con IA,
responde automáticamente las consultas de pedidos y alerta al equipo
cuando hay un reclamo urgente."

*Opcional: mostrar el flujo en n8n 10 segundos — sin explicar nodo por nodo.*

---

## Minuto 2:00 — 6:00: LA DEMO EN VIVO

*"Voy a mostrar cómo funciona en tiempo real."*

**Acciones (no hablar mucho — dejar que el sistema hable):**
1. Abrir el email/formulario/WhatsApp
2. Enviar el mensaje de prueba en vivo
3. Cambiar a n8n y mostrar la ejecución en curso
4. "Acá pueden ver cómo el sistema procesa el mensaje paso a paso"
5. Mostrar el resultado: la respuesta al cliente / la alerta / el registro en Sheets

*Si hay tiempo: mostrar un segundo caso (reclamo o caso de error).*

---

## Minuto 6:00 — 7:00: EL REPORTE Y EL DASHBOARD

*"Además de responder en tiempo real, el sistema genera su propio reporte."*

1. Abrir el dashboard de Google Sheets
2. Mostrar las métricas de la semana
3. Si hay tiempo: mostrar el reporte automático que llegó el último lunes

---

## Minuto 7:00 — 9:00: LOS NÚMEROS

*"El sistema funciona. Ahora les cuento cuánto vale."*

Citar de MEMORIA (no leer de la pantalla):

"Esta semana el sistema procesó **[X] mensajes**.
Respondió automáticamente **[Y]** — el **[Z]%** del total — sin que nadie
tuviera que hacer nada.
Eso equivale a **[X horas]** de trabajo que el sistema hizo solo.
El ROI anual calculado es de **[X]%**."

---

## Minuto 9:00 — 10:00: EL SIGUIENTE PASO

*"Si siguiera desarrollando este sistema los próximos 3 meses, haría:"*

**Completar:**
- La próxima mejora concreta: [descripción]
- Por qué: [beneficio específico]
