## El día de hoy

**Formato:**
- 10 minutos de presentación por participante
- 2 minutos de preguntas del grupo
- El docente mantiene el tiempo

**Lo que el evaluador quiere ver:**
1. El sistema funcionando en vivo (no un video, no una captura)
2. Un número real de impacto (horas, pesos, ROI)
3. Que el participante entiende lo que construyó

**Lo que NO importa:**
- Que el sistema sea perfecto
- Que el código sea sofisticado
- Que use todas las herramientas posibles
