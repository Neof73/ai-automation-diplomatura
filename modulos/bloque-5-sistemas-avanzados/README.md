# 🔴 Bloque 5 — Sistemas inteligentes avanzados (Semanas 21–25)

**Objetivo:** Llevar el agente del Bloque 4 al siguiente nivel. Memoria persistente que aprende del negocio, orquestación de múltiples agentes especializados, procesamiento de imágenes y audio, blindaje del sistema para producción real y monetización de la automatización como servicio.

## Clases

| Semana | Título | Tipo | Caso Lumina |
|---|---|---|---|
| 21 | Memoria persistente y RAG: el agente que aprende de tu negocio | Práctica | Base de conocimiento de catálogo y políticas |
| 22 | Sistemas multiagente: orquestar equipos de IA especializados | Práctica | Agente clasificador + agente post-venta + agente marketing |
| 23 | IA multimodal: imágenes, PDFs y voz en tus flujos | Práctica | Cliente envía foto de producto dañado, factura en PDF, audio de WhatsApp |
| 24 | Seguridad, privacidad y producción: blindar el sistema | Práctica | Sistema Lumina auditado, documentado y listo para escalar |
| 25 | De sistema a producto: monetizar y escalar tu automatización | Workshop | Lumina como servicio replicable para otras tiendas |

## Entregable del bloque

**Sistema avanzado en producción:** el agente del negocio propio, extendido con al menos una capacidad avanzada (RAG, multimodal o multiagente), con un plan documentado de seguridad y una propuesta de valor para monetizarlo como servicio.
